1) Difference between HTML Tags and Elements?

-> HTML tags- Tags are the starting and ending part of an HTML element. They begin with “<” symbol and end with “>”. Whatever written inside < and > are called tags. For instance: -- <b> <b>
-> HTML elements – Elements enclose the contents in between the tags. They consist of some kind of structure. It generally consists of a start tag, content and end tag. For instance: -- < Content >


2) Explain < /br > tag?

-> The < br > tag in HTML document is used to create a line break in a text. It is an empty tag which means it does not need end tag like < /br >. It is generally used in poem and address where the division of line is necessary


3) Explain how many types of list we have?

-> In HTML we have three types of lists
  1. Ordered List
    2. In ordered lists we following types of lists
    1. Numbered List(1,2,3)
    2. Uppercase Letters(A,B,C)
    3. Lowecase Letters(a,b,c)
    4. Uppercase Roman Letters(I,II,III)
    5. Lowercase Roman Letters(i,ii,iii)
  2. UnOrdered List
    3. In unordered lists we following types of lists
    • cicle List(1,2,3)
    • disc type list
    • square type list
    • none type list
  3. Description List
    4. A Description List is a list of item which contains a description
